# Taaleem - Digital Learning Platform

Taaleem is a premium, state-of-the-art digital learning platform built with **Next.js 14** and styled using a custom **dark glassmorphism** system (inspired by Lumina Learning design guidelines). The platform is tailored for students and teachers, featuring full client-side authentication, real-time filtering, and interactive, live classroom tools.

---

## 🌟 Key Features

### 1. 🔒 Premium Portal Authentication
- Secure client-side login guard protecting all platform subpages.
- Interactive user menu dropdown allowing easy logout.
- Persistent user sessions using local storage.

### 2. ⚡ Interactive Student Dashboard (Hero & Stats)
- **Interactive Search**: Filters course topics and lectures dynamically as you type.
- **Micro-interactions**: Glassmorphic quick action cards with hover effects.
- **Academic Progress Rings**: Dynamic progress indicators for completion rates, live attending classes, and study hours.

### 3. 📹 Live Interactive Classroom Hub
- **Simulated Streaming Feed**: Real-time mock stream of live class sessions.
- **Classroom Sketchpad/Whiteboard**: Fully functional drawing canvas supporting brush colors, brush size, eraser, and clear options.
- **Group Class Chat Room**: Send messages during class that appear in the live chat log.

### 4. 📖 Recorded Lecture Archives & Notes
- Archive with filter controls for subjects (Math, Physics, Biology, History) and class grades.
- **Lecture Notes Manager**: Opens a simulated video player with a side-by-side note-taking notebook to write and save notes during the video.

### 5. 📅 Timetable & Schedule Tracker
- Tabbed schedule view categorized by weekdays (Monday - Friday).
- Active live status indicators, completed indicators, and "Remind Me" alert toggles.

### 6. 📝 Assignments Portal
- Workspace categorized by pending, submitted, and graded tasks.
- **File Submission Modal**: Drag and drop mock files and add text notes to submit homework.

### 7. 🏆 Achievements & Leaderboard
- Class leaderboard ranking Rajesh Kumar at #2 in Class 10-A.
- Unlocked badges showcase (with social sharing copy links).
- Interactive weekly point claims.

## 🛠️ Tech Stack

* **Framework**: [Next.js 14](https://nextjs.org/) (App Router)
* **Library**: [React 18](https://react.dev/)
* **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
* **Icons**: [Lucide React](https://lucide.dev/)
* **Components**: [Radix UI](https://www.radix-ui.com/) & [shadcn/ui](https://ui.shadcn.com/)

## 🚀 Getting Started

To run the project locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/iamkaifyyy/Taaleem.git
   cd Taaleem
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

4. **Open in browser**:
   Navigate to [http://localhost:3000](http://localhost:3000) (or the port specified in terminal).
